Algorithms Used

Backtracking: Used to solve the maze by exploring all possible paths from the start to the end, backtracking when hitting a dead end.

Divide and Conquer: For larger mazes (10x10+), this algorithm:

  • Divides the maze into smaller sections
  • Creates strategic checkpoints between start and end
  • Solves each section independently
  • Combines the solutions for better performance
